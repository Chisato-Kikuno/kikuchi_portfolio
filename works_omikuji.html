おみくじアプリ

概要

JavaScriptを使った簡単なおみくじができるアプリケーションです。おみくじを引くをクリックすると、ランダムでおみくじ結果が表示されます。

制作ポイント

大吉や凶、大凶は出にくいように確立をかえています。またおみくじを引いたときにドキドキできるように、結果表示までディレイさせたり、音がでるように設定しました。

使用ツール

Visual Studio Code

使用素材

いらすとや 効果音ラボ
